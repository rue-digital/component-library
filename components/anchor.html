Anchor

<a>

The anchor tag defines a hyperlink. Its href attribute indicates the link's destination. The target attribute specifies where to open the linked document.

  • _self is the default and opens in the same frame
  • _blank opens the link in a new tab
  • _parent opens the link in the parent frame
  • _top opens the link in the full body of the window

Skip Link

Skip links allow people using keyboards, screen readers, and other assistive technologies to reach the main content faster. Commonly, the skip link is the first interactive element on a page, but it is hidden until it receives focus.

To be accessibly for all users, the skip link must:

  • be accessibly via keyboard navigation
  • is visually distinctive when focused
  • should set focus to the main content when activated

To add a skip link, add an anchor element as the first child of the header and link it to the main content. Make sure href value of the skip link matches the id attribute of the main element. Here's an example:

<header>
<a href="#main" class="skip">Skip to main content</a>
...
</header>
<main id="main">
...

Temporarily hidden skip links can also be used by hiding it off screen with CSS, then allowing it to be visible when it recieves keyboard focus. Hiding the skip link with display:none or hidden will remove it from keyboard navigation. Instead, make the link the same color as the background or transparent. Use CSS transitions so it remains visible on screen for more time, if necessary.

Base Document Structure

<header>

The header tag acts as a container for introductory content, including logos, navigation menu, search form, and author name.

<header> <!-- Introductory or navigational content --> </header>

Headers can be used page-wide for the whole site or within long articles, giving the content structure. They can be nested under article tags, aside, nav, or section.

<main>

The main tag specifies the main content of a document. Content inside main should be unique to the document and shouldn't repeat across documents.

<main> <!-- Main content --> </main>

<footer>

The footer tag defines the footer for a document or section. It typically contains supplementary information that isn't main content such as copyright information, contact information, back to top links, and sitemap. Multiple footer elements can exist in one document.

<footer> <!-- Copyright information --> </footer>

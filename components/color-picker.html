Color Picker

<input type="color">

Input elements of type color allow users to select a color

Color picker will default to #000000 (black). To customize its value on load, use the value attribute. Add a label tag for best accessibility practices.

<label for="favorite">your favorite color:</label>
<input type="color" value="#FFC0CB"/>

A color's input event is called everytime its color changes. The change change event is fired when the user clicks out of the color picker.

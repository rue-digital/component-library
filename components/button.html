Button

<button>

The button tag creates a clickable button on a document.

Always specify the type attribute for a button element. Buttons can be of type button, reset, and submit. Buttons with type "button" creates a clickable button that does not have any default behavior. Submit button submits a form and reset button resets all form fields.

A clickable button looks something like this:

<button type="button">Click Me!</button>

or

<button type="submit">Submit</button>

or

<button type="reset">Clear</button>

Other common attributes include:

  • autofocus, boolean that specifies if an element should recieve focus when the page loads
  • disabled, specifies that a button should be disabled
  • form, specifies which form the button belongs to
  • value, text that establishes the initial value for a button
